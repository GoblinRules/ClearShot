# ClearShot v1.0.2

- **Fixed Check for Updates** — Moved to Settings → About tab with SSL fallback for bundled exe compatibility
- **Consolidated Settings** — Help, About, and Check for Updates are now in the Settings window under a new "About" tab
- **Fixed Tray Icon Size** — Icon path now correctly resolves in PyInstaller-bundled executables via `sys._MEIPASS`
- **Cleaner Tray Menu** — Simplified context menu: Capture options, Open Folder, Settings, Help/About, Exit
