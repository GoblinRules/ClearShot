# ClearShot v1.0.1

## 🔧 Changes

- **Improved Tray Icon Quality** — Now uses pre-rendered icon pack PNGs at exact pixel sizes (24, 32, 48, 64, 128, 256px) for pixel-perfect icons instead of scaling down from a large source
- **Added Help Dialog** — In-app usage guide accessible from the tray menu (❓ Help)
- **Added Check for Updates** — Check for new releases directly from the app (🔄 Check for Updates)
- **Added GitHub link** to the About dialog
- **Updated installer** — Correct publisher info, GitHub URLs, license display, and version info
